"# Mov-Rec-pre2000s" 
The dataset has been drawn from https://grouplens.org/datasets/movielens/ [older dataset: size 5mb]
The program makes use of libraries seaborn, numpy, pandas, requests, bs4.
The scrapping has been done on the web-browser results using BeautifulSoup.
This project makes use of dataset of old movies, hence new movie results would not be available directly, however you can use the datasets available on the aforesaid website (for free) which is nearly 250mb in size.
This project is totally for educational purpose and the primary contributor(s) have no involvement in any of its commercial use.
